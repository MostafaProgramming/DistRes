How To Demonstrate

  1. Open this page in two browser windows.
  2. Log in as different users, for example alice and ben.
  3. Click Subscribe in each window that should receive events.
  4. Start reading in one window, then request Write Lock in the other.
  5. Stop reading and watch the queued writer receive exclusive access.
  6. Save the file and show the EVENT UPDATE notification in the other window.
